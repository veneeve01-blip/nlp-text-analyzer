# NLP Text Analyzer

## 📌 Description
A simple Natural Language Processing project that analyzes text.

## ⚙️ Features
- Tokenization
- Stopword removal
- Displays important words

## 🛠️ Tech Used
- Python
- NLTK

## ▶️ How to Run
1. Install nltk: pip install nltk
2. Run main.py
3. Enter a paragraph
